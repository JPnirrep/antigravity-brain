# 🧠 Antigravity Brain : Évolution Majeure - Projet SENTINEL-G & HIVE-G

## 📅 Date : 13 Mars 2026

## 🛡️ 1. Système Immunitaire : Sentinel-G
Mise en place d'une architecture de sécurité "Zero-Trust" pour l'importation et l'exécution d'agents tiers.
- **Zone de Quarantaine** : Tout nouveau code (GitHub) est isolé dans `./quarantine/`.
- **Audit Statique (Sentinel)** : Analyse des patterns dangereux (os.system, subprocess, eval).
- **Certification (Registrar)** : Signature cryptographique (SHA-256) des scripts validés stockée dans Firestore et en local (`registry/trust_store.json`).
- **Barrière d'Exécution (Gatekeeper)** : Seuls les scripts certifiés et non-altérés peuvent être lancés.

## 💰 2. Gestion Budgétaire (Pillage PaperClip)
Intégration d'une protection financière atomique :
- Chaque agent possède désormais un quota financier (`budget_limit`) défini dans son certificat.
- Le **Gatekeeper** bloque automatiquement l'exécution si le budget est épuisé.

## 🐝 3. Orchestration : HIVE-G (The Hive)
Évolution de l'orchestrateur vers un modèle de "Ruche" asynchrone et intelligente.
- **Atomic Checkout** : Utilisation de transactions Firestore pour garantir qu'une tâche n'est traitée que par une seule butineuse.
- **Context Modes (Thin/Fat)** : Injection dynamique du contexte pour optimiser les coûts et la précision.
- **Swarm Execution** : Lancement parallèle de multiples mini-agents pour des tâches de recherche et d'analyse web.
