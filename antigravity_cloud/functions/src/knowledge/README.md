# 🧠 Module : KLEIA-UP (Base de Connaissances)

## 📌 Vision Globale
Ce dossier contient l'audit croisé et la base de connaissances dédupliquée du projet **KLEIA-UP** (évolution de Souffles Positifs). Il est structuré pour une consommation **token-frugale** via Firebase.

## 🛠 Structure de la Mémoire
- **`index_map.json`** : Point d'entrée unique (Indexation/Routage).
- **`data_vault.yaml`** : Mémoire dédupliquée (13 exercices validés).
- **`sources_registry.yaml`** : Pont vers les sources originales sur Google Drive.
- **`agent_logic.md`** : Skills, identité de l'agent et méthodologie.

## 🤖 Instructions de Routage
Ne charge jamais l'intégralité du module. Utilise `index_map.json` pour identifier le fragment pertinent (Identity, Stress, Body, Content).
